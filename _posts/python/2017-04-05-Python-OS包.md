- os.path.abspath(path)

  返回`path `的绝对路径

- os.path.dirname(path)

  返回`path`所在的路径

下面举个例子来说明：

有一个文件config.py，所在的路径是`/home/work/web_app/config_files/`，文件内容为：

```python
import os
a = os.path.abspath(__file__)
b = os.path.dirname(a)
c = os.path.dirname(b)
print a
print b
print c
```

打印出来的内容为：

```shell
/home/work/web_app/config_files/config.py
/home/work/web_app/config_files
/home/work/web_app
```
