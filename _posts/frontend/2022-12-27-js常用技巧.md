#### dict object loop

array 可以用 map, forEach, dict object 怎么办?

```js
var myObject = { a: 1, b: 2, c: 3 };

Object.keys(myObject).forEach(function (key, index) {
  myObject[key] *= 2;
});

console.log(myObject);
// => { 'a': 2, 'b': 4, 'c': 6 }
```

#### Input

##### onfocus cursor end

```js
onFocus={(e) => {
  const val = e.target.value;
  e.target.value = "";
  e.target.value = val;
}}
```
