##### Icon

首页左上角的 icon, 可以这样写:

```js
import logo from "./logo.png";

<div
  className="logo"
  style=\{\{
    backgroundImage: `url(${logo})`,
    backgroundRepeat: "no-repeat",
    backgroundSize: "contain",
  \}\}
```

css 可以写固定 pixel 的宽高

```css
.logo {
  float: left;
  width: 40px;
  height: 40px;
  margin: 16px 24px 16px 0;
}
```
